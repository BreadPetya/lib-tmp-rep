# lib-tmp-rep
## Установка UV
```
pip install uv
```
## Добавьте UV в PATH
```
source ~/.cargo/env
```
# Использование UV
## Создание виртуального окружения
```
source .venv/bin/activate
```
# Установка пакетов
## Установка конкретных пакетов
```
uv pip install pandas numpy
uv pip install .
```
# Запуск проекта
```
python main.py
```
